# 校园探索 · 设计文档

日期: 2026-07-01
状态: Draft

## 概述

在博客中新增项目 3「江农校园探索」—— 2D 像素风的江西农业大学校园地图，用户可以通过键盘控制角色在校园里行走，查看建筑信息，帮助新生快速熟悉校园。

## 技术选型

| 项 | 选择 |
|---|---|
| 框架 | Vue 3 + Vite（与博客一致） |
| 地图渲染 | Canvas / Div Grid（像素瓦片） |
| 角色控制 | 键盘 WASD / 方向键 |
| 数据来源 | 用户提供的校园地图 + OpenStreetMap 标注 |
| 部署 | 现有博客的子路由 `/campus` |

## 功能清单

### V1 核心功能
- [x] 像素瓦片地图渲染（多种地形: 道路、建筑、草坪、水域）
- [ ] 角色移动（WASD / 方向键，碰撞检测）
- [ ] 建筑名称标签绘制在地图上
- [ ] 走到建筑旁自动弹出信息面板
- [ ] 地点总览（分类列出所有建筑，点击导航）

### V2 新生引导
- [ ] 进入欢迎提示
- [ ] 推荐路线（新生报到路线、生活路线等）
- [ ] 路线标记在地图上

### V3 增强功能
- [ ] 搜索建筑
- [ ] 当前位置显示
- [ ] 建筑收藏/标记

## 地图数据

瓦片编码:
- 0: 空地/广场 (#a8d5ba)
- 1: 道路 (#ccc2a7)
- 2: 建筑 (#7c8a9e) 
- 3: 草坪 (#7dbf6b)
- 4: 水域 (#4d9de0)
- 5: 入口 (#8b6f47)
- 6: 树木 (#4a7c3f)

建筑数据结构:
```
{
  id: string
  name: string
  desc: string
  x/y: 瓦片坐标
  w/h: 占地面积
  color: 标签颜色
  category: 分类
  tips?: 实用贴士
}
```

## 组件结构

```
Campus.vue          — 主页面：地图渲染 + 角色控制 + 信息面板
campus-map.js       — 地图瓦片数据 + 建筑列表
```
