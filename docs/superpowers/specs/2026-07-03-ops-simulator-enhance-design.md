# 运维模拟平台 - 沉浸专业版 设计规格

**日期**: 2026-07-03
**项目**: 个人博客 - 项目2（运维模拟平台）
**版本**: v1.0

## 1. 概述

对现有的网络运维模拟游戏进行全面的UI/交互升级，打造"赛博运维控制台"风格的沉浸式运维模拟平台。保留现有关卡内容和核心游戏逻辑，在视觉、交互、流程、反馈等方面全面提升。

## 2. 设计理念

「赛博运维控制台」— 深蓝黑基调 + 霓虹青/绿高光 + 玻璃拟态面板，打造"真实运维监控中心"的沉浸感。所有图标使用 lucide-vue，零 emoji。

## 3. 模块设计

### 模块一：整体架构与布局

**游戏主界面布局**（三栏 + 底终端）:
- 顶部状态栏：关卡进度、计时器、错误次数、菜单
- 左侧工具面板：运维工具箱（可折叠）
- 中央拓扑图区：可切换视角（全局/机房/设备）
- 右侧信息面板：设备详情/告警列表/性能指标（Tab 切换）
- 底部终端区：可展开/收起

**三个页面阶段**:
1. 关卡选择页
2. 游戏主界面
3. 复盘页

**响应式**:
- 桌面（>1200px）：完整三栏布局
- 平板（768-1200px）：左右面板可折叠
- 手机（<768px）：Tab 切换 + 终端全屏

### 模块二：拓扑图多视角系统

**三种视角**:
1. 🌐 全局拓扑：完整网络拓扑图，设备SVG图标，链路数据流动效
2. 🗄️ 机房机柜：42U机柜正视图，设备堆叠，端口灯，跳线连接
3. 🔍 设备特写：单设备面板大图，端口列表，状态指标

**交互**:
- 点击设备 → 切换到特写视图
- 双击设备 → telnet/ssh 自动连接
- 悬停链路 → 带宽/延迟/丢包气泡
- 滚轮缩放 + 拖拽平移
- 视角切换有平滑过渡动画

**设备图标**: 全部使用 lucide-vue（Monitor, Server, Switch, Router, Shield 等）

### 模块三：终端增强与工具面板

**终端升级**:
- 深灰黑背景 + CRT扫描线效果
- 语法高亮（成功绿/错误红/警告黄/信息青/IP紫/端口橙）
- Tab 自动补全
- Ctrl+R 历史搜索
- 上下键翻历史
- 长输出可折叠

**左侧工具面板**（垂直排列，可折叠）:
- Ping 快捷测试
- Traceroute 路由追踪
- DNS 查询
- 远程连接（telnet/ssh）
- 端口扫描
- 配置管理
- 诊断工具
- 笔记

**右侧信息面板**（三 Tab）:
- 📡 设备详情：基本信息、端口状态、CPU/内存
- 🚨 告警列表：实时告警滚动，级别颜色区分
- 📈 性能指标：带宽趋势、流量排行、错误计数

**快捷命令栏**: 根据当前设备动态显示相关命令按钮

### 模块四：诊断修复流程与评分系统

**分层诊断面板**:
- 按 OSI 七层模型分组（物理层/数据链路层/网络层/应用层）
- 每层可展开/折叠
- 可标记"已排除"选项
- 答错扣机会，3次错误触发提示

**引导式修复**:
- 3个修复方案选项（最优/次优/错误）
- 终端自动执行修复命令（打字机效果）
- 执行后需玩家自行验证
- 不同选择影响评分

**星级评分**（1-3星）:
- ⏱️ 用时 30%
- 🎯 诊断准确率 30%
- 🔧 修复方案 20%
- 📝 命令效率 20%

**顶部状态栏**: 关卡名 · 计时器 · 错误次数 · 命令数 · 菜单

**成功/失败反馈**:
- 成功：绿色脉冲 + SYSTEM RESTORED + 评分卡片滑入
- 失败：红色警告 + 提示查看提示

### 模块五：复盘系统与进度保存

**顶部评分卡片**: 星级 + 四维度分项得分 + 总分

**操作时间线**: 玩家所有操作按时间顺序记录，关键节点高亮

**最优路径对比**: 表格对比最优路径 vs 玩家路径，每步给出评价

**知识点扩展卡片**: 根据关卡涉及的技术点展开知识讲解

**进度保存**（localStorage）:
- 已完成关卡 + 星级记录
- 最佳用时、最少命令数
- 总游戏时长、累计完成关数
- 关卡选择页显示进度和星级

### 模块六：项目对接与动画

**项目详情页对接**:
- 项目2的"在线预览"按钮跳转游戏
- 按钮样式升级更醒目
- 封面区域增加互动感

**关卡选择页重做**:
- 按难度分组（教学/基础/进阶）
- 卡片显示星级、状态（完成/进行中/锁定）
- 顶部总进度条
- 锁定关卡灰色半透明

**过场动画**:
- 关卡开始：任务简报淡入
- 关卡完成：庆祝动画 + 星级点亮

**微交互动效**:
- 设备状态变化：颜色渐变 + 光晕脉冲
- 告警出现：滑入 + 震动
- 命令执行：打字机效果
- 面板展开：平滑过渡
