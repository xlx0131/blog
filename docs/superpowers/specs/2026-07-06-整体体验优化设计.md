# 网站整体体验优化设计文档

> 版本：v1.0
> 日期：2026-07-06
> 状态：待实现

## 一、概述

### 1.1 目标

对个人博客网站进行整体体验优化，包括导航系统、动画过渡、性能优化、移动端适配和交互规范统一，全面提升用户体验，同时保持像素风格的一致性。

### 1.2 范围

- 导航系统：面包屑导航、文章页侧边目录
- 动画与性能：页面切换过渡、图片懒加载、性能优化
- 移动端：底部导航栏、侧边抽屉菜单
- 交互规范：统一像素风按钮/卡片/过渡效果

### 1.3 设计原则

- **像素风格统一**：所有交互效果符合像素风美学，无模糊、无发光、无圆角
- **性能优先**：动画不影响页面性能，使用 CSS transform 和 opacity
- **渐进增强**：核心功能在低端设备也能正常使用，动画是加分项
- **可访问性**：键盘可操作、颜色对比度足够、减少动效选项

## 二、导航系统优化

### 2.1 面包屑导航

**组件名**：`Breadcrumb.vue`

**位置**：页面内容区域顶部，导航栏下方

**样式**：
- 像素风：米黄色背景（#fffaef）+ 2px 黑色边框 + 3px 像素阴影
- 字体：等宽字体，12px
- 分隔符：`/` 或像素箭头

**结构规则**：
- 首页：不显示面包屑
- 一级页面（/about, /projects, /archive 等）：`首页 / 关于`
- 二级页面（/projects/:id, /article/:id 等）：`首页 / 作品集 / 项目名称`

**交互**：
- 每一级都可点击跳转
- 当前页不可点击，颜色加深表示选中
- 文字过长时截断，显示省略号

**路由映射表**：
| 路径 | 面包屑 |
|------|--------|
| /about | 首页 / 关于 |
| /projects | 首页 / 作品集 |
| /projects/:id | 首页 / 作品集 / 项目名 |
| /archive | 首页 / 归档 |
| /article/:id | 首页 / 归档 / 文章标题 |
| /bookmarks | 首页 / 书签 |
| /bookmarks/:id | 首页 / 书签 / 书签标题 |
| /network-game | 首页 / 网络运维游戏 |

### 2.2 文章页侧边目录

**组件名**：`ArticleToc.vue`

**位置**：文章详情页右侧，fixed 定位，跟随滚动

**功能**：
- 自动提取文章中 H2、H3 标题生成目录树
- 滚动页面时自动高亮当前阅读章节
- 点击目录项平滑滚动到对应位置
- 目录超出视口高度时内部可滚动
- 移动端隐藏（屏幕宽度 < 1024px）

**样式**：
- 像素风卡片：米黄背景 + 2px 边框 + 像素阴影
- 当前章节：左侧 2px 蓝色竖条 + 文字加粗
- H2 项：左对齐，主层级
- H3 项：缩进 16px，次层级
- 滚动条：像素风格

**实现思路**：
- 使用 `IntersectionObserver` 检测标题可见性
- 或监听 scroll 事件，计算当前滚动位置对应的标题
- 目录数据在组件挂载时从 DOM 中提取

## 三、动画与性能优化

### 3.1 页面切换过渡动画

**实现方式**：Vue Router `<Transition>` 包裹 `<RouterView>`

**动画效果**：
- 前进（新页面进入）：新页面从右侧 16px 淡入滑入
- 后退（返回上页）：新页面从左侧 16px 淡入滑入
- 动画时长：300ms
- 缓动函数：`cubic-bezier(0.25, 0.8, 0.25, 1)`

**像素风约束**：
- 不使用 filter、blur、backdrop-filter 等效果
- 位移值为整数像素
- 透明度过渡线性
- 整体感觉干脆利落

**前进/后退判断**：
- 使用 `useRouter` 监听路由变化
- 维护历史栈，比较新旧路由层级判断前进后退
- 或简单处理：统一淡入淡出，不区分方向

### 3.2 图片懒加载

**组件名**：`LazyImage.vue`

**功能**：
- 图片进入视口前不加载
- 使用 `IntersectionObserver` API 检测
- 加载前显示像素风占位符
- 加载完成后淡入（150ms opacity 过渡）
- 加载失败显示错误占位符
- 支持原生 `loading="lazy"` 兜底

**占位符样式**：
- 背景：米黄色（#f5f0e8）+ 像素网格纹理
- 中央：像素加载动画（4个像素方块轮流闪烁）
- 尺寸：根据图片原始比例计算，防止布局跳动
- 使用 `aspect-ratio` CSS 属性保持比例

**使用方式**：
```html
<LazyImage src="xxx.jpg" alt="描述" :ratio="16/9" />
```

### 3.3 性能优化清单

- [x] 路由懒加载（确认所有路由都用动态导入）
- [ ] 大组件按需引入（NetworkGame 相关组件只在游戏页面加载）
- [ ] GSAP ScrollTrigger 优化（确保组件卸载时正确销毁）
- [ ] 图片懒加载（新增 LazyImage 组件）
- [ ] 减少不必要的重绘重排（transform/opacity 优先）

## 四、移动端优化

### 4.1 移动端底部导航栏

**组件名**：`MobileNav.vue`

**显示条件**：屏幕宽度 < 768px

**样式**：
- 固定在屏幕底部
- 像素风：米黄背景 + 顶部 2px 黑色边框
- 高度：56px
- 4 个等宽 Tab

**Tab 项**：
1. 首页 — 房子图标 + "首页"文字
2. 作品集 — 文件夹图标 + "作品"文字
3. 关于 — 用户图标 + "关于"文字
4. 更多 — 菜单图标 + "更多"文字（点击弹出侧边抽屉）

**交互**：
- 当前页面高亮（蓝色文字 + 上边框）
- 点击切换页面有轻微按压反馈
- 图标 20px，文字 10px

### 4.2 侧边抽屉菜单

**组件名**：`SideDrawer.vue`

**触发方式**：
- 顶部导航栏汉堡菜单按钮（桌面端也可保留）
- 底部导航栏"更多"按钮（移动端）

**样式**：
- 从右侧滑入
- 宽度：80vw（最大 320px）
- 像素风：米黄背景 + 左侧 2px 边框 + 左侧像素阴影
- 遮罩层：半透明黑色背景，点击关闭

**菜单项**：
- 首页
- 作品集
- 归档
- 书签
- 关于
- 网络运维游戏（彩蛋）
- 分隔线
- 主题切换（预留）

**交互**：
- 滑入动画：300ms，从右侧 100% → 0
- 遮罩淡入：200ms
- 点击菜单项后自动关闭抽屉
- 支持 Esc 键关闭

### 4.3 响应式断点

- **桌面**：≥ 1024px — 完整导航 + 侧边目录
- **平板**：768px ~ 1024px — 顶部导航，隐藏目录
- **手机**：< 768px — 汉堡菜单 + 底部导航

## 五、统一像素风交互规范

### 5.1 按钮规范

**默认状态**：
- 背景：#fffaef（米白）
- 边框：2px solid #161310（深黑）
- 阴影：3px 3px 0 0 #161310
- 文字：#161310，等宽字体，13px，字重 600
- padding：10px 16px

**Hover 状态**：
- 背景：#f2ead6（稍深米黄）
- 位移：translate(-1px, -1px)
- 阴影：4px 4px 0 0 #161310
- 过渡：150ms all

**Active 状态**（按下）：
- 位移：translate(2px, 2px)
- 阴影：1px 1px 0 0 #161310
- 过渡：50ms all

**禁用状态**：
- 背景：#d4c8b8（灰）
- 文字：#8a7a68（淡）
- 阴影：3px 3px 0 0 #8a7a68
- 光标：not-allowed

**主按钮（Primary）**：
- 背景：#2e5dd6（蓝色）
- 文字：#fffaef（米白）
- 其他规则同上

### 5.2 卡片规范

**默认状态**：
- 背景：#fffaef
- 边框：2px solid #161310
- 阴影：4px 4px 0 0 #161310

**Hover 状态**：
- 位移：translate(-2px, -2px)
- 阴影：6px 6px 0 0 #161310
- 过渡：200ms all

### 5.3 过渡规范

- **快速交互**（按钮、标签）：150ms
- **中等过渡**（卡片、面板）：200ms
- **页面切换**：300ms
- **缓动函数**：`cubic-bezier(0.25, 0.8, 0.25, 1)`
- **禁用效果**：不使用 blur、glow、backdrop-filter 等非像素效果
