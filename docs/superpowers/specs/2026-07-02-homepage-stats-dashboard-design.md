# 首页数据看板设计

## 概述

在首页第二屏（主卡片区域下方）新增三个数据统计卡片，展示项目数、博客数和 GitHub 累计星星数。

## 布局

位于现有卡片（头像 + 导航链接）下方，三个数据项横排一行：

```
┌─────────────────────────────────────┐
│              头像                    │
│           许立鑫                     │
│    写代码 · 做设计 · 思考人生        │
│    [归档]   [项目]   [关于]          │
│  ─────────────────────────────────  │
│   📦 项目    📝 博客    ⭐ 星星     │
│    N 个       N 个      N 个        │
└─────────────────────────────────────┘
```

## 数据源

| 数据 | 来源 | 方式 |
|------|------|------|
| 项目数 | `src/views/Projects.vue` 的 `projects` 数组长度 | 直接导入数据，或硬编码常量 |
| 博客数 | `src/views/Archive.vue` 的 `articles` 数组长度 | 直接导入数据，或硬编码常量 |
| GitHub 星星 | GitHub REST API | 通过 `https://api.github.com/users/xlx0131/repos` 获取所有仓库，累加 `stargazers_count` |

## 交互效果

- 数据看板跟随卡片一同淡入（`card-inner.in` 之后延迟出现）
- 数字从 0 递增计数到目标值（使用 animejs，因为项目已依赖该库）
- GitHub 星星在页面加载时通过 fetch 请求，缓存到 sessionStorage 避免重复请求

## 样式

- 延续现有暗黑主题配色（`#0d1117` 背景、`#e6edf3` 文字、`#34d399` 强调色）
- 每个数据项显示为一个小卡片/统计块，带图标和标签
- 响应式：大屏三列横排，小屏（<640px）单列竖排

## 实现文件

- 仅修改 `src/views/Home.vue`，无需新增组件或文件
